# GitHub Repository Analysis

放送大学「プログラミング応用」の最終レポート用プロジェクトです。

## Features

- GitHub APIから人気Pythonリポジトリ100件を取得
- CSV形式で保存
- Star数ランキングの可視化
- Star数とFork数の相関分析

## Technologies

- Python 3.13
- pandas
- matplotlib
- seaborn
- requests

## Graphs

- Top 10 repositories by stars
- Correlation between stars and forks
